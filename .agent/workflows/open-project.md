# Open List App Project

## Steps

1. Open the project folder in Finder:
```bash
open -a "Finder" /Users/philreed/.gemini/antigravity/scratch/list-app
```

2. (Optional) Start the development server:
```bash
cd /Users/philreed/.gemini/antigravity/scratch/list-app && npm run dev
```

3. (Optional) Open the app in browser at http://localhost:3000
